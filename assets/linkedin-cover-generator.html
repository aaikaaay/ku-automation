📸 How to Export as PNG (1128 x 191 px)

Method 1 — Browser DevTools:
1. Open this file in Chrome
2. Right-click on your preferred cover → Inspect
3. In DevTools, right-click the .cover element → Capture node screenshot
4. This exports at exact dimensions!

Method 2 — Screenshot:
• macOS: Cmd + Shift + 4 → drag selection over the cover
• Crop/resize to exactly 1128 x 191 px if needed
