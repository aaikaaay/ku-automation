📸 How to Export as PNG

1. Open this file in Chrome/Safari
2. Right-click on your preferred logo
3. Select Inspect Element
4. In DevTools, right-click the element → Capture node screenshot

Or use a screenshot tool:
• macOS: Cmd + Shift + 4 then drag over the logo
• Resize to exactly 300x300px in Preview or any image editor
